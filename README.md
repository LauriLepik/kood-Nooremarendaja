These projects were developed as part of **[kood/Nooremarendaja](https://kood.tech/kood-nooremarendaja/)**, a **free, EU-funded, 5-month intensive coding program** provided by **kood/J√µhvi**.

### About kood/J√µhvi üè´

**kood/J√µhvi** is a novel coding school in Estonia that operates on a radical **"no teachers, no classes"** philosophy. Based on the international **01 Edu System**, the methodology emphasizes:

- **Self-Directed Learning**: Students progress at their own pace through a gamified curriculum.
- **Peer-to-Peer Collaboration**: All learning happens through group work, code reviews, and solving complex problems together (mimicking real-world engineering teams).
- **Practical Focus**: The curriculum is 100% hands-on coding, designed to build industry-ready skills in a short timeframe.

### The [kood/Nooremarendaja](https://kood.tech/kood-nooremarendaja/) Program üéì

This **5-month intensive program** (funded by **[NextGenerationEU](https://kood.tech/meist/toetused/)**) is designed to rapidly upskill participants for Junior Developer roles. The curriculum is structured into four distinct modules:

1. **Java Sprint (3 Weeks)**: Intensive immersion in programming basics, setting up the development environment, and solving algorithmic challenges.
2. **Back-end Development (7 Weeks)**: Deep dive into server-side logic, database management, architecture design, and REST APIs.
3. **Front-end Development (3 Weeks)**: User interface development using JavaScript and DOM manipulation to create interactive web applications.
4. **Practical Project / Internship (8 Weeks)**: A capstone experience where all skills are applied to build a complete web application with real data and multiple user roles.

Beyond technical syntax, the program instills critical professional habits:

- **Real-World Simulation**: No artificial deadlines or exams; success depends on managing time and delivering working software against functional requirements.
- **Soft Skills**: The peer-learning model forces constant communication, conflict resolution, and feedback exchange (code reviews are mandatory).
